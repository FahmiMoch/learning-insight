Frontend dari proyek **AI Learning Insight** yang menampilkan progres belajar, tipe belajar, insight, dan rekomendasi hasil analisis model Machine Learning.

### Teknologi yang digunakan:
- React
- TailwindCSS
- Vite

### Fitur:
- Mengintegrasikan API backend untuk menampilkan data real-time
- Menggunakan Context API untuk state global (Auth + Insight)
- UI sesuai desain Figma
- Interaktif dan responsif di berbagai ukuran layar
